FAQ

Quick answers to common questions.

Is this a finished product?

This is the Replit-ready landing page. Next we’ll add auth, transaction storage, analytics, and the AI assistant via the OpenAI API.

Can we add a dashboard quickly?

Yes. The next step is FastAPI + SQLite (or Postgres), sign-up/sign-in forms, and protected routes like /app.

How will the AI assistant work?

We’ll send aggregated data (categories, totals, timeframe) to the OpenAI API and return actionable advice—without sending unnecessary personal details.
